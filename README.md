# TSun-LookUp

## Overview

TSun-LookUp is a web application for searching user data using a phone number. It provides a user-friendly interface with multiple language support and dark/light mode options.

## Features

- Phone number search functionality
- Multi-language support
- Dark/light mode toggle
- Feedback form with Formspree integration
- Responsive design

## Third-Party Libraries and Styles

### UI Components
- Radix UI components (Accordion, Alert Dialog, Avatar, etc.)
- Tailwind CSS for styling
- Lucide React for icons
- Framer Motion for animations
- Shadcn UI components (Button, Card, Input, etc.)

### Utilities
- i18next for internationalization
- React Hook Form for form handling
- Zod for form validation
- Sonner for toast notifications
